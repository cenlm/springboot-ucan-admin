# springboot-ucan-admin

#### Description
基于RBAC3的权限管理系统，Springboot版本。 该系统使用Shiro作为用户认证与授权管理框架，实现了基于RBAC权限管理功能。 技术栈:Springboot+ Shiro + Mysql + LayUI + Freemarker。 欢迎交流学习！
